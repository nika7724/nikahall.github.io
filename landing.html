CMS_Project

Building an Application for rendering data
from Contentful Management System via API.

About_The_Project

Agile_Methodology - Scrum

My goal for this project is to create an application capable of displaying various types of content, such as blogs, product information, documentation, and more, sourced from a Content Management System (CMS). Additionally, I integrated this CMS data into another application that I developed later on (see Portfolio2). Furthermore, my application is equipped to automatically upload content from the CMS to the customer page.

  • Frontend

Prototype

In my application, content is stored in the Contentful database and fetched via API in the Svelte application I developed. This content is then rendered using iframe in another application for customer access.

Data_Strcture_Of_CMS

I fetched data from the CMS, but due to its intricate structure, I needed to extract and manage it systematically. To facilitate this, I created an ArrayList for mapping and data extraction. Given the complexity of the nested array, it was essential to create a new list to effectively handle the data.
