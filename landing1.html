About_The_Application

In this application, customers can register and log in, and view their list of customers. Additionally, customers can utilize functions such as "Create," "Update," and "Delete." The data is successfully retrieved from the CMS via API and displayed within the application. The data is presented on the homepage and other relevant pages.

Security

Using BCryptPasswordEncoder transforms a user's password into a fixed-length string using a one-way hash function, ensuring that the original password cannot be reversed. BCryptPasswordEncoder is extremely secure for brute force attacks. The password was originally set as "nika", but it was encoded using a one-way hash function.
