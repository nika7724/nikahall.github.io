CNN_Body & Kernel

In this process, 0 represents black and 1 represents white. The objective is to identify various features within an image, such as edges, vertical lines, horizontal lines, bends, etc. These features are represented in a feature map, making them easier to analyze or classify. Kernel weights are randomly provided when the model runs. The sum of each pixel in the image and each element of the kernel is calculated after applying convolution. ReLU (Rectified Linear Unit) is applied, wherein negative values in the feature maps are replaced by zero, while positive values remain unchanged.

CNN_Max Pooling & Flatten

Converting different patterns, such as horizontal to vertical, into a format suitable for inputting into a densely connected neural network requires a fully connected layer to accept only one-dimensional data (a vector).
